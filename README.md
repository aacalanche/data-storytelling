## About the project
This project analyzes a dataset of video game sales and ratings.
The dataset is sourced from Kaggle: [Video Game Sales with Ratings](https://www.kaggle.com/datasets/rush4ratio/video-game-sales-with-ratings/data).

This dataset contains 16,791 rows, about 6,900 of which contain complete records. A complete record contains:
- Name (Categorical)
- Platform (Categorical)
- Year of Release (Numerical)
- Genre (Categorical)
- Publisher (Categorical)
- North America Sales (Numerical)
- EU Sales (Numerical)
- Japan Sales (Numerical)
- Other Sales (Numerical)
- Global Sales (Numerical)
- Critic Score (Numerical)
- Critic Count (Numerical)
- User Score (Numerical, after cleaning)
- User Count (Numerical)
- Developer (Categorical)
- Rating (Categorical)

Totaling to 16 columns, 6 categorical and 10 numerical.
