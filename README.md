## About the project

GameVault is a Django web application for browsing, managing, and analyzing video game sales data. It combines a CRUD interface for individual game records with an analytics dashboard that summarizes sales, review scores, genres, platforms, and regional performance.

## Features

- Browse a paginated list of games
- View full details for each game record
- Create, edit, and delete games through Django forms
- Seed the database from a cleaned CSV dataset
- Explore an analytics dashboard powered by `pandas` and `Chart.js`
- Manage data through the Django admin
